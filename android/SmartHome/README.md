## SmartHome 智能家居

基于51单片机和安卓APP的智能家居系统.

本项目利用[零度创意工作室](http://www.hx-wl.com.cn)推出的51-WIFI开发板开发了一个含温湿度、烟雾报警、GSM模块、语音控制的智能家居系统。
